Compatibilité en répétabilité

Voici comment les résultats précédents sont utilisés concrètement. Supposons que vous utilisiez une méthode qui a déjà été validée après une étude INTER-laboratoire. Vous avez donc accès à l'écart type de répétabilité sr de cette méthode. Cela vous donne l'avantage suivant : en réalisant plusieurs mesures à l'aide de cette méthode, vous pouvez déterminer si ces mesures sont compatibles (en conditions de répétabilité), et donc savoir si pouvez raisonnablement avoir confiance en ces mesures. Pour cela, on utilise la limite de répétabilité, c'est-à-dire l'écart maximal entre deux mesures pour qu'elles soient considérées comme compatibles avec une probabilité de 95%. Pour tous les laboratoires, sauf le labo 3, les deux mesures sont compatibles : la valeur retenue sera donc leur moyenne. Pour le labo 3, on suit le logigramme, en supposant ici que nous sommes dans le cas d'un mesurage coûteux : on obtient une troisième mesure, celle-ci figure dans le tableau de l'étude INTER-laboratoire : y3=1,9177 mol/L. Par conséquent :
ymax-ymin=2,3311-1,7975=0,5336 mol/L

Cette valeur doit être comparée à la nouvelle limite de répétabilité : CR0,95=3,3⋅9,2⋅10-3=0,030 mol/L. Nous sommes toujours au-desssus, donc les 3 valeurs ne sont toujours pas compatibles. Il faut donc prendre la médiane, c'est-à-dire la valeur centrale des trois mesures, soit y3=1,9177 mol/L.
